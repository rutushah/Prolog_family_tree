# Family Tree Program in Prolog

This project implements a simple family tree in Prolog using facts and rules.

## Files
- `family_tree.pl` - Prolog facts and rules
- `report.docx` - Assignment report

## Features
- Parent relationship
- Child relationship
- Sibling relationship
- Grandparent relationship
- Cousin relationship
- Recursive descendant relationship
